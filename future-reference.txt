---------------------------------------------------------------------------------------

1. Initial Setup
Use these to start a fresh project on any computer.

Create Virtual Environment

# Windows
python -m venv venv
venv\Scripts\activate

# Mac/Linux
python3 -m venv venv
source venv/bin/activate

Install Dependencies

# Install all required packages at once (assuming you have requirements.txt)
pip install -r requirements.txt

# OR Install manually (The complete list)
pip install fastapi uvicorn python-dotenv pydantic langchain langchain-community langchain-google-genai langchain-pinecone langchain-text-splitters pinecone-client pypdf langgraph duckduckgo-search rank_bm25 mcp

Fix Namespace Issues (If "ModuleNotFoundError: langchain.retrievers" appears)

# Force reinstall to fix broken dependencies
pip install -U langchain langchain-community langchain-core

---------------------------------------------------------------------------------------

2. Configuration (.env)

Create a file named .env and ensure it has these keys.

GOOGLE_API_KEY=AIzaSy...
PINECONE_API_KEY=pc-sk...

---------------------------------------------------------------------------------------

3. Running the Application

Start the Backend API (FastAPI)
Runs the server at http://localhost:8000

python main.py

Ingest Data (Upload PDF to Pinecone)
Run this only when you add a new my_cv.pdf file.

python ingest.py

Check Google Models
Run this if you get "404 Model Not Found" errors.

python check_models.py

6. Key Troubleshooting Notes

"Dimension Mismatch" Error:
Cause: You used OpenAI (1536 dims) settings but Google Embeddings (768 dims).

Fix: Delete Pinecone Index -> Create New Index -> Dimensions: 768.

"Unexpected token L" (MCP Error):
Cause: You have a print() statement in server_mcp.py.

Fix: Delete or comment out all print() statements outside of @tool functions.

Claude Desktop Plugin Missing:
Cause: Config file named claude_desktop_config.json.txt.

Fix: Enable "Show File Extensions" in Windows Explorer and rename it to .json.

Google 404 Error:
Cause: Using generic gemini-1.5-flash.

Fix: Switch to specific version gemini-1.5-flash-001 or gemini-1.5-flash-latest.
